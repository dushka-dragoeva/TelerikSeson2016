We are given a school. In the school there are 
classes of students. Each class has a set of teachers.
Each teacher teaches, a set of disciplines. Students have a name and unique class number. 
Classes have unique text identifier. Teachers have a name. 
Disciplines have a name, number of lectures and number of exercises.

Both teachers and students are people. 
 
Students, classes, teachers and disciplines could have optional comments (free text block).
