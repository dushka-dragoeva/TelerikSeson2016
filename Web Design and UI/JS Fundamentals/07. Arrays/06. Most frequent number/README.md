### Problem 6. Most frequent number
*	Write a script that finds the most frequent number in an array. 

_Example:_

|                  input                |    result   |
|---------------------------------------|-------------|
| **4**, 1, 1, **4**, 2, 3, **4**, **4**, 1, 2, **4**, 9, 3 | 4 (5 times) |
