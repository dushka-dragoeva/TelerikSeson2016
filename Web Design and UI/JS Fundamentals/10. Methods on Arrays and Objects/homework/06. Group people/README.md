### Problem 6. Group people
*   Write a function that groups an array of persons by first letter of first name and returns the groups as a JavaScript Object
    *   Use **Array#reduce**
    *   Use **only array methods** and no regular loops (for, while)

_Example:_

        result = {
            'a': [{
                firstname: 'Asen',
                /* ... */
            }, {
                firstname: 'Anakonda',
                /* ... */
            }],
            'j': [{
                firstname: 'John',
                /* ... */
            }]
        };
